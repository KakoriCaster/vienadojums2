Apraksts

Šī ir vienkārša tīmekļa lietotne, kas palīdz aprēķināt kvadrātvienādojuma diskriminantu, balstoties uz lietotāja ievadītajiem koeficientiem.

Lietošana

Atveriet index.html jebkurā tīmekļa pārlūkprogrammā.

Ievadiet trīs skaitliskās vērtības laukos a, b un c, kas ir kvadrātvienādojuma koeficienti.

Noklikšķiniet uz pogas Aprēķināt.

Rezultāts – diskriminanta vērtība – tiks parādīts zem ievades laukiem.

Formulas aprēķins

Diskriminanta formula:

Kur:

a – kvadrātiskā locekļa koeficients

b – lineārā locekļa koeficients

c – brīvais loceklis

Prasības

Tīmekļa pārlūkprogramma (Chrome, Firefox, Edge u.c.).

Fails

index.html – satur HTML, CSS un JavaScript kodu, kas nepieciešams lietotnes darbībai.
